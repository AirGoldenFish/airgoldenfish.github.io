# md5
## md5(s)\[a:b]=='xxxx'
直接爆破就完事了。自己写了遍代码，没有开多线程，速度也挺快的。
```python
import hashlib
#····························设置数据
knowMD5=""
start=0
amount=1     	 #爆破出来的个数
#····························完成
end=start+len(knowMD5)
def md5(text):
    return hashlib.md5(str(text).encode('utf-8')).hexdigest()
if __name__ =='__main__':
     i=0
     themd5=0
     while(i<amount):
         if(md5(themd5)[start:end]==knowMD5):
             print(themd5)
             print(md5(themd5))
             i+=1
         themd5+=1
```
